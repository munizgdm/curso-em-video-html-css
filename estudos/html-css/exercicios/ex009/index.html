Tipos de Listas

Listas Ordenadas

Para criar uma ordered list, vamos usar a tag <ol> </ol>

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

A tag <ol> </ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parãmetro são:

1 - Valor padrão. Cria listas numeradas. Ex: 1, 2, 3, 4, ...

A - Cria listas alfabéticas em maiúsculas. Ex: A, B, C, D, ...

a - Cria listas alfabéticas em minúsculas. Ex: a, b, c, d, ...

I - Cria listas com algarismos romanos em maiúsculas. Ex: I, II, III, IV, ...

i - Cria listas com algarismos romanos em minúsculas. Ex: i, ii, iii, iv, ...

Listas não Ordenadas

Para criar uma unordered list, vamos usar a tag <ul> </ul>

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

O marcador padrão é a bolinha preta totalmente preenchida (circle), mas existe a opção de configurar a propriedade type da tag <ul> </ul> com os seguintes valores:

disc - Padrão, uma bola preta totalmente pintada

circle - Uma bola com uma borda preta e sem preenchimento

square - Um pequeno quadrado preto totalmente pintado

Misturando as coisas

Podemos também crar listas mistas, configurando listas dentro de outras listas. Veja o exemplo a seguir:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Lista de definições

É como se fosse um dicionário, temos os termos e as suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos.

Toda lista de definições está dentro de uma tag <dl> </dl> (definition list) Cada termo é um <dt> </dt> (definition term) e cada descrição é um <dd> </dd> (definition description). Assim como os itens da lista, essas duas últimas tags possuem fechamento opcional, segundo a referência oficial HTML5.
